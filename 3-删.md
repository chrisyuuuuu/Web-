### 1.1 删除指定文档
	- 标准语法:
		DELETE /index/type/id
	- 例子
		DELETE /goods/fruit/1
		操作结果如下:
		{
		  "_index" : "goods",
		  "_type" : "fruit",
		  "_id" : "1",
		  "_version" : 14,
		  "result" : "deleted",------这指明了是已经删除了
		  "_shards" : {
			"total" : 2,
			"successful" : 1,
			"failed" : 0
		  },
		  "_seq_no" : 13,
		  "_primary_term" : 1
		}
